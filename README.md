# safher_women_safety_ai

Instructions to run //local machine only

//Windows 


1) pull the main in a single folder.
2) Open the folder in any code editor.
3) exectution conditions:- system must contain latest/base Python and node.js
4) open the terminal (ctrl+~)
5) next are exact copy paste commands to be entered
6) cd safety_intelligence_system
7) python run_servers.py
8) AI modules activation done
9) next are steps to open the frontend servers
10) open new terminal without closing/terminating the first one
11) next are exact copy paste commands to be entered
12) cd map
13) cd client
14) npm start
15) Commands are done . Terminal will return a localhost link which needs to be pasted  in any browser in  the machine module
17) Ensure fast internet to ensure all routes are rendered by frontend as well as route graph from open street map by the Ai.
18) Best results are limited to delhi ncr.
